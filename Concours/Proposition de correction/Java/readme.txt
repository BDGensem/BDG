Pour coder en java, le concours avait plusieurs contraintes :

- la classe a le même nom que l'exercice
- les bibliothèques utilisables sont java.util.*, et java.io.*
- les entrées sont lues avec des scanners :
    - Scanner sc = new Scanner(System.in);
    - String ligne = sc.nextLine();
- la sortie est lue avec :
    - System.out.println("Sortie");
